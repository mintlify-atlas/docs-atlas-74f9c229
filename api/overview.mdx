The `mazegen` package provides maze generation, solving, ASCII visualization, and file output. It is distributed as a Python wheel and has no external runtime dependencies.

## Installation

```bash
pip install mazegen-1.0.0-py3-none-any.whl
```

## Quick import

```python
from mazegen import (
    MazeGenerator,
    MazeError,
    generate_output,
    render_maze,
    render_solve,
    render_generation,
    render_path,
    Color,
)
```

## Building the package yourself

The repository ships a `pyproject.toml`. To produce the wheel:

<Steps>
  <Step title="Install build">
    ```bash
    pip install build
    ```
  </Step>
  <Step title="Build the wheel">
    ```bash
    python3 -m build -w
    ```
    The `.whl` file is written to the `dist/` directory.
  </Step>
  <Step title="Install the wheel">
    ```bash
    pip install dist/mazegen-1.0.0-py3-none-any.whl
    ```
  </Step>
</Steps>
